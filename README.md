# Template 3 Data Science Portfolio

This is a ready-to-upload GitHub Pages portfolio website for Ashwini Balasubramanian.

## Files

- `index.html` — website content
- `style.css` — design and responsive layout
- `script.js` — menu, scroll animations, and counter animation
- `resume.pdf` — add your resume with this exact filename

## How to publish

1. Create a GitHub repository named `ashbala155.github.io`.
2. Upload all files from this folder.
3. Upload your resume and name it `resume.pdf`.
4. Go to Settings → Pages.
5. Choose Deploy from branch → main → /root.
6. Open `https://ashbala155.github.io`.

## Customize

Edit `index.html` to add your LinkedIn URL, GitHub project links, live demo links, and final resume.
